# TFY4220 — Faste stoffers fysikk

Interaktiv studieguide for [TFY4220](https://www.ntnu.no/studier/emner/TFY4220)
ved NTNU: **[fastestoffer.martinsundal.no](https://fastestoffer.martinsundal.no)** — moduler med quiz, flashcards,
formelsamling og begreper.

Bygget på [StudyCompanion](https://github.com/MartinSA04/StudyCompanion);
dette repoet inneholder kun innhold (`content/`).

## Kjør lokalt

```bash
pnpm install
pnpm dev
```
